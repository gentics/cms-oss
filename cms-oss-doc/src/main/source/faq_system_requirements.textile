h2. System Requirements

These System requirements are examples and must be adapted for each project and the existing IT infrastructure

The continuously updated list of compatible components like OSs, Browsers, Databases can be found "here":gcn_compatibility_list.html.

endprologue.

h3. Server Requirements

h4. Minimum Hardware Requirements

* x86 Dual Core 2+ GHz CPU
* 4 GB Memory
* 15 GB Disk space

h4. Recommended Hardware Requirements

* x86 Quad Core 2+ GHz CPU
* 8 GB Memory
* 100 GB Disk space

h4. CMS Server - Operation System

* "One of the supported 64 Bit Linux Distributions":gcn_compatibility_list.html
* "One of the supported CMS Databases (MySQL,MariaDB)":gcn_compatibility_list.html

NOTE: We suggest the use of 64Bit systems because a 64Bit system can address more memory (RAM). This proved to be future-proof in many cases.

h3. Client Requirements

h4. Client Browser

* Screen resolution of at least 1024x768
* Recommended screen resolution 1280x1024 or higher
* enabled JavaScript

h4. Client Systems

* Microsoft Windows
* Linux
* MacOS

h3. Supported Repositories

* Oracle MySQL
* Microsoft SQL Server
* Oracle Database Server
* Gentics Mesh
