h2. Overview of discontinued Gentics CMS Features

This page provides a list of *discontinued* Gentics CMS features.

endprologue.

h3. What are discontinued features?

Discontinued features are features, that have been developed for older versions
of Gentics CMS. They are not improved or maintained anymore. Most of
those features won't have any visible impact when changed. Nonetheless you
should not change or remove those features from your node.conf file.

WARNING: If you find features listed here in your node.conf configuration file
you don't need to worry. Do *not change or remove* discontinued from your
@node.conf@ as this might affect your system.  Gentics customer support will
propose changes if neccessary.
