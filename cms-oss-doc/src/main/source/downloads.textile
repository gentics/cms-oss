h2. Downloads

This page lists all downloads for Gentics CMS

endprologue.

All our downloads are managed via maven. You can browse through the maven repository.

h3. Packages

* The "cms-server":https://repo.gentics.com/service/rest/repository/browse/maven/com/gentics/cms-oss/cms-oss-server/ is the JAR file that must be launched to start Gentics CMS.

* The "cms-js-lib (gcnjsapi)":https://repo.gentics.com/service/rest/repository/browse/maven/com/gentics/cms-oss/cms-js-lib/ contains the Gentics CMS JavaScript API downloadable as ZIP file.
