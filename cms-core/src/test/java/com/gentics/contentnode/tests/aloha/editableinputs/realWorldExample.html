FAQ: Was ist ein Node Tags?
Node-Tags sind Platzhalter für alle bearbeitbaren Elemente einer Seite. Jedes Element wird durch einen Node-Tag definiert. Node-Tags basieren auf sogenannten Node-Tagtypen, die das Aussehen und die Funktionsweise von Node-Tags bestimmen.
