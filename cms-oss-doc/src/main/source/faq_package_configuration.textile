h3. Debian

h4. Pre Requirements

The following packages have to be installed:
<shell>
  # apache2
  apt-get install apache2

  # php
  apt-get install php5
</shell>

You can either install MySQL Server from the distribution or install MySQL Server manually using the latest MySQL Server binaries:

Distribution MySQL:
<shell>
  apt-get install install mysql-server-5.5
</shell>

Binary MySQL:
* Download the latest MySQL Server 5.6 binary from "mysql.com":http://dev.mysql.com/downloads/mysql/
* Install the MySQL Server using the provided "documentation":http://dev.mysql.com/doc/refman/5.6/en/installing.html

h3. SUSE Linux Enterprise Server

h4. Pre Requirements

The following packages have to be installed:
<shell>
  # apache2
  zypper install apache2

  # php
  zypper install php53 php53-curl  apache2-mod_php53 php53-mysql
</shell>

If you want to allow apache access though the firewall you can invoke the following commands:
<shell>
  sysconf_addword /etc/sysconfig/SuSEfirewall2 FW_CONFIGURATIONS_EXT apache2
  sysconf_addword /etc/sysconfig/SuSEfirewall2 FW_CONFIGURATIONS_EXT apache2-ssl
  rcSuSEfirewall2 restart
</shell>

You can either install MySQL Server from the distribution or install MySQL Server manually using the latest MySQL Server binaries:

Distribution MySQL:
<shell>
  zypper install mysql mysql-client
</shell>

Binary MySQL:
* Download the latest MySQL Server 5.6 binary from "mysql.com":http://dev.mysql.com/downloads/mysql/
* Install the MySQL Server using the provided "documentation":http://dev.mysql.com/doc/refman/5.6/en/installing.html
